# 简介
基于vue+webpack+eslint+vuex+axios框架

# 安装
npm install

# 启动
npm start

# 对src源码进行检查
npm run lint

# 打包
npm run build

#  目录结构说明
* .babelrc 里面配置es6转es5
* .gitignore 是忽略上传git的文件
* .package.json 项目依赖的npm包的列表
* src 里面就是项目的源文件目录
* dist 项目编译打包后文件目录
* webpack.config.js webpack配置文件
* .eslintrc.js ESLINT配置文件
* postcss.config.js postcss配置文件 配置自动补全css前缀

# 源码文件src详解
* views vue模板存放目录
* app.js vue入口文件
* index.html 入口html模板
* router 路由配置文件
* components 组件存放目录
* sass sass文件存放目录
* plugins 插件存放目录
* store store文件存放目录
* util util文件存放目录
* imgs 图片存放目录
